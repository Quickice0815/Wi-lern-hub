# WI-Lern-Hub

Interaktive Lernplattform für Wirtschaftsinformatik – Prof. Dr. Eduard Gerhardt.

Fünf Lernmodule, direkt im Browser spielbar. Von der Startseite aus wählt man ein
Themengebiet per Klick auf eine Karte aus und gelangt direkt (ohne Seitenwechsel,
als Ansicht innerhalb derselben App) in das jeweilige Programm:

- **📚 Die 5 Artikel** – Fachartikel mit Zusammenfassung, Quiz und Anwendungsaufgabe
- **🔗 ERM-Trainer** – Aus Text ein Entity-Relationship-Modell bauen
- **🔢 Historische Zahlensysteme** – Römisch, Ägyptisch, Babylonisch, Maya
- **🧙‍♂️ PAP-Quest** – Programmablaufpläne & Kontrollstrukturen als Quest
- **🗄️ SQL-Filterung trainieren** – SAP-Geschäftspartner mit WHERE/AND/OR/LIKE filtern

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech-Stack

React + TypeScript + Vite + Tailwind CSS.
